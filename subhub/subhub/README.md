# The Sub Hub — Website

Everything is self-contained. Just open `index.html` in a browser and it works.
The page currently shows labeled **placeholder** images so nothing is blank.
Replace them with real photos whenever you're ready.

## Folder layout
```
subhub/
├── index.html          <- the page
├── README.md           <- this file
└── images/
    ├── badge.png        (your logo — already final)
    ├── hero-sub.jpg     1400 x 1000  — big stacked sub, dark background
    ├── fav-italian.jpg   600 x 600   — The Italian
```

## How to swap in a real photo
1. Find a photo (see free sources below).
2. Rename it to **exactly** the filename above (e.g. `fav-italian.jpg`).
3. Drop it into the `images/` folder, replacing the placeholder.
4. Refresh the page. Done — no code editing needed.

Tip: match the shape roughly (the listed size is a guide). The page crops
photos to fit automatically, so exact pixels don't matter — just avoid using a
tall portrait photo where a wide one is expected.
